## 名称
视点管理

## 英文名
ViewPoint

## 缩略图
![](./logo.png)

## 版本
前端：v1.0.2

后台：v1.0.0 

## 描述
支持用户添加任意观察角度的视点，并捕捉当前镜头状态为缩略图，添加到视点列表中，备注视点标签名称，方便后期检索。后续点击查看具体视点时，可直接定位到对应的空间位置，并同步开启室内模式、地下模式等功能，同时绑定图层的显示和关闭功能。

## 数据规范要求
无
