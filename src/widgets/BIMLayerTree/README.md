BIM图层树

## 英文名
BIMLayerTree

## 描述
对BIM模型按类别、楼层生成树结构，并进行显隐控制，可对构件统计、颜色设置、定位

## 数据规范要求
BIM模型需要入库，并且转成3dtiles时要写进入库表里面的类别、楼层属性
